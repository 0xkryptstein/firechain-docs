# Getting Started

This guide will help you set up your environment and begin interacting with the FireChain network.

---

## Step 1: Connect to FireChain

To interact with FireChain, your wallet or application needs access to a network node. This is done using an RPC endpoint.

Follow the guide below to configure your wallet:

👉 [RPC endpoints](./developers/rpc-endpoints)

---

## Step 2: Get Test Tokens

Before you can send transactions or deploy contracts, you will need FIRE tokens on the test network.

You can request free test tokens here:

👉 [Faucet](./developers/faucet)

These tokens are used only for development and testing.

---

## Step 3: Start Interacting with the Network

Once connected and funded, you can begin using FireChain:

- Send transactions between wallets  
- Deploy and interact with smart contracts  
- Build and test decentralized applications
