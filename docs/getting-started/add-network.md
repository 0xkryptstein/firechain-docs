# Add Network

To use FireChain, you need to add the network to your wallet.

---

## Network Details

Use the following official network configuration:

```text
Network Name: Rinia
RPC URL: https://rpc-rinia.firestation.io
Chain ID: 917
Currency Symbol: FIRE
Block Explorer URL: https://rinia.firescan.io/
```

## General Wallet Flow

Most wallets follow a similar process:

1. Open your wallet
2. Go to network settings
3. Choose to add a custom network
4. Paste the official values above
5. Save and switch to the network

## Important

Always use official network details from FireChain documentation.

If a value does not match the information above, do not add it until verified.
