# First Transaction

Once you have added the network and bridged FIRE into FireChain, you are ready to make your first transaction.

---

## Common First Actions

Typical first actions include:

- Sending FIRE to another wallet
- Importing a token
- Swapping on DonaSwap
- Interacting with a deployed contract
- Reading data from the block explorer

---

## Basic Checklist

Before making a transaction, confirm:

- You are on the correct FireChain network
- You have FIRE in your wallet
- The address you are interacting with is official or verified
- You understand what the transaction does

---

## Good Practice

Always start with a small transaction first.

This helps confirm:

- Wallet setup is correct
- Network connection is working
- Token balances display properly
- The destination address is valid
