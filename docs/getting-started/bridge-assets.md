# Bridge Assets

The FireChain bridge lets you move supported assets from connected networks into FireChain.

---

## Bridge URL

Use the official bridge only:

`https://rinia-bridge.firestation.io`

---

## Supported Networks

You can currently bridge from:

- Ethereum Sepolia
- Base Sepolia
- Optimism Sepolia
- BNB Smart Chain Testnet

---

## General Bridge Flow

1. Open the official bridge
2. Connect your wallet
3. Select the source network
4. Select FireChain as the destination
5. Choose the token and amount
6. Confirm the transaction
7. Wait for completion and finalization

---

## Before Bridging

Make sure that:

- Your wallet is connected to the correct source network
- You are using the correct FIRE token contract for that network
- You have enough gas on the source network
- You are using the official FireChain bridge
