# Get FIRE

To interact with FireChain, you need FIRE tokens.

At this stage, FIRE is available across supported network environments connected through the bridge.

## How to Use FIRE

The usual flow is:

1. Hold FIRE on a supported external network
2. Use the bridge to move FIRE into FireChain
3. Interact with applications and contracts on Rinia
