# Quick Start

This guide will help you begin using FireChain in a few simple steps.

---

## Step 1: Create or Open a Wallet

Use a supported wallet such as:

- MetaMask
- Rabby
- WalletConnect-supported wallets

If you do not already have a wallet, start here:

👉 [Create Wallet](./create-wallet)

---

## Step 2: Add the FireChain Network

Add the FireChain network to your wallet using the official network details.

👉 [Add Network](./add-network)

---

## Step 3: Get FIRE

To interact with the network, you need FIRE tokens on a supported network before bridging them into FireChain.

👉 [Get FIRE](./get-fire)

---

## Step 4: Bridge Assets

Bridge FIRE from a supported network into FireChain.

👉 [Bridge Assets](./bridge-assets)

---

## Step 5: Make Your First Transaction

Once your wallet is connected and funded, you can begin interacting with the network.

👉 [First Transaction](./first-transaction)
