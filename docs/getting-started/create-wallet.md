# Create Wallet

Before using FireChain, you need a compatible wallet.

---

## Supported Wallet Types

FireChain works with EVM-compatible wallet flows, including:

- MetaMask
- Rabby
- WalletConnect-compatible wallets

---

## Important Security Rules

When creating a wallet:

- Store your recovery phrase offline
- Never share your private key
- Never paste your recovery phrase into websites or chats
- Double-check extensions and downloads before installing them

---

## Recommended Setup

If you are new, the easiest starting point is MetaMask.

👉 [MetaMask Setup](../wallet/metamask)

If you already use a different EVM wallet, you can still connect by adding the FireChain network manually.
