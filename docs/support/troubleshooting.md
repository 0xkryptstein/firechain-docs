# Troubleshooting

This page covers general issues users may encounter while using FireChain.

---

## Wallet Does Not Connect

Check that:

- You added the correct network
- The Chain ID is `917`
- The RPC URL is correct
- Your wallet is switched to Rinia

---

## Token Does Not Appear

If a token balance is not visible:

- Confirm the token exists in your wallet
- Import the token manually if needed
- Verify the contract address from the docs

Useful page:

👉 [Import Token](../wallet/import-token)

---

## Bridge Problems

If bridging fails or appears delayed:

- Confirm the source network
- Confirm the FIRE token address
- Confirm you have enough gas
- Wait for the transaction to finalize before retrying

Useful page:

👉 [Bridge Troubleshooting](../bridge/troubleshooting)

---

## Contract Interaction Fails

Check that:

- The contract address is correct
- Your wallet is on the correct network
- You have enough FIRE for gas
- The contract function and parameters are correct
