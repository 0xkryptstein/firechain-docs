# Import Token

If a token does not appear automatically in your wallet, you can import it manually using its contract address.

---

## Common Tokens

### DonaSwap Token

`0x5dfCf2674af93B7155b65c7084dce9353d704915`

### WFIRE

`0xA00F48aed2Ac0314d0D5e46EddCc201df77B8a99`

---

## Before Importing

Always confirm the token contract against the official FireChain documentation.

Never import a token from a copied message or random social media post without verification.

---

## General Wallet Flow

1. Open your wallet
2. Select import token
3. Paste the contract address
4. Confirm the token details
5. Add the token
