FireChain supports standard EVM wallet flows.

This section explains how to connect supported wallets and import tokens correctly.

## Common Wallet Actions

You may need to:

- Create a wallet
- Add the FireChain network
- Switch to the correct network
- Import a token
- Confirm transactions
- Review balances and approvals
