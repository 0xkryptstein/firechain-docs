# WalletConnect

WalletConnect can be used when a FireChain-compatible application supports it.

---

## Basic Flow

1. Open the supported FireChain application
2. Choose WalletConnect
3. Scan the QR code or approve the session
4. Confirm the selected wallet account
5. Switch to the correct network when prompted

---

## Important

Always make sure that:

- The application is official
- The requested network is correct
- The contract interaction matches what you expect

---

## Best Practice

Never approve blind signatures or unexpected contract interactions.

Always verify the application and address first.
