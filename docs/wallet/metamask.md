# MetaMask

MetaMask is one of the easiest ways to connect to FireChain.

---

## Add FireChain to MetaMask

Use the following network details:

```text
Network Name: Rinia
RPC URL: https://rpc-rinia.firestation.io
Chain ID: 917
Currency Symbol: FIRE
Block Explorer URL: https://rinia.firescan.io/
```

## Setup Flow

1. Open MetaMask
2. Go to network settings
3. Add a custom network
4. Paste the official values
5. Save the network
6. Switch to Rinia

After Setup

Once connected, you can:

## Receive FIRE
- View transactions
- Interact with contracts
- Use the bridge and ecosystem applications
