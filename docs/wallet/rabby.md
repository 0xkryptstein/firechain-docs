# Rabby

Rabby can also be used with FireChain through a custom EVM network configuration.

---

## Add the Network

Use the official FireChain network values:

```text
Network Name: Rinia
RPC URL: https://rpc-rinia.firestation.io
Chain ID: 917
Currency Symbol: FIRE
Block Explorer URL: https://rinia.firescan.io/
```

## General Flow

1. Open Rabby
2. Add a custom network
3. Enter the official values
4. Save and switch to FireChain

## Important

Before signing any transaction:

- Verify the contract address
- Confirm the selected network
- Confirm the explorer or application is official
