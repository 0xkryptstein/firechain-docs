# Verify Addresses

Before interacting with any token or contract, verify the address first.

---

## What to Verify

Always verify:

- Token addresses
- Router addresses
- Factory addresses
- Liquidity manager addresses
- Explorer URLs
- Bridge URLs

---

## Trusted Sources

Use these docs as your primary source of truth.

You can also cross-check with the official FireChain explorer:

`https://rinia.firescan.io/`

---

## Important Reminder

Do not trust:

- Screenshots without context
- Edited messages
- Random forwarded Telegram posts
- Unofficial spreadsheets
- DMs claiming to provide updated addresses
