# Block Explorer

The FireChain block explorer lets you inspect wallets, contracts, tokens, and transactions on the network.

---

## Official Explorer

`https://rinia.firescan.io/`

---

## What You Can Inspect

You can use the explorer to view:

- Wallet addresses
- Token contracts
- Smart contracts
- Transactions
- Internal transfers
- Contract code and verification status
