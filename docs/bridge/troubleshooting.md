# Bridge Troubleshooting

If you run into issues while bridging, use this checklist.

---

## Common Issues

### Wrong Source Network

Make sure your wallet is connected to the network you selected in the bridge.

### Wrong Token

Verify that you are using the official FIRE token address for that network.

### Not Enough Gas

Keep enough native gas on the source network for both approval and bridge submission.

### Transaction Pending

Wait for confirmations and check the explorer before retrying.

### Wrong Website

Only use the official bridge:

`https://rinia-bridge.firestation.io`
