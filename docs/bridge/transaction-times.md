# Transaction Times

Bridge completion times can vary.

---

## What Affects Timing

Bridge times can depend on:

- Source network confirmation speed
- Congestion on the source network
- Relayer or bridge processing steps
- Destination network settlement timing

---

## General Guidance

If your bridge transaction appears delayed:

- Check the source network transaction in your wallet
- Look up the transaction in the source network explorer
- Confirm you used the correct network and token
- Wait for the bridge process to finalize before retrying

---

## Important

Do not repeatedly resend the same bridge transaction unless you are sure the previous one failed.

If you are unsure, verify the transaction status first.
