# How to Bridge

This guide explains the practical flow for bridging assets into FireChain.

---

## Step-by-Step

1. Open the official bridge  
   `https://rinia-bridge.firestation.io`

2. Connect your wallet

3. Choose the source network:
   - Ethereum Sepolia
   - Base Sepolia
   - Optimism Sepolia
   - BNB Smart Chain Testnet

4. Select the token and amount

5. Approve the token if required

6. Confirm the bridge transaction

7. Wait for the bridge to complete

8. Switch your wallet to FireChain if needed and verify receipt

---

## Recommended Practice

Use a smaller test amount first when bridging for the first time.

This helps confirm that:

- The wallet is connected correctly
- The selected source network is correct
- The token is the official deployment
- The bridge flow is working as expected
