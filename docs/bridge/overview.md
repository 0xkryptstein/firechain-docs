# Bridge Overview

The FireChain bridge connects FireChain with supported external networks so users can move supported assets into the ecosystem.

---

## Official Bridge

`https://rinia-bridge.firestation.io`

---

## What the Bridge Enables

The bridge is used to:

- Move FIRE from supported networks into FireChain
- Connect liquidity and user flows across networks
- Bring assets into the FireChain environment for on-chain use

---

## Supported Networks

The bridge currently supports:

- Ethereum Sepolia
- Base Sepolia
- Optimism Sepolia
- BNB Smart Chain Testnet
