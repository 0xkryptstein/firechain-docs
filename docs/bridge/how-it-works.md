# How It Works

The FireChain bridge lets users move supported assets from external networks into FireChain.

---

## Basic Flow

1. Connect your wallet
2. Select the source network
3. Select the asset you want to bridge
4. Enter the amount
5. Approve and confirm the transaction
6. Wait for bridge processing and final settlement

---

## What to Check Before Confirming

Always confirm:

- The source network is correct
- The token contract is correct
- You have enough gas on the source network
- The destination is the official FireChain environment
- You are using the official bridge URL

---

## Finalization

Bridge transactions can take time depending on network conditions and confirmation requirements.

Always wait for confirmation before retrying a bridge transaction.
