# Welcome to FireChain

FireChain is a blockchain ecosystem built for scalable decentralized applications, smart contract deployment, digital assets, and cross-chain interoperability.

The documentation is designed to help both users and developers get started quickly, connect wallets, bridge assets, explore official addresses, and build on the network.

---

## Current Status

FireChain is currently live in its active testing phase.

At this stage, users can:

- Add the network to supported wallets
- Bridge FIRE from supported test networks
- Explore contracts and addresses on the block explorer
- Interact with DonaSwap on Rinia
- Build and test smart contracts
