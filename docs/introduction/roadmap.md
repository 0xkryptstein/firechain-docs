# Roadmap

FireChain is being developed in phases, with a focus on building reliable infrastructure first and expanding the ecosystem over time.

---

## Current Phase

The current phase focuses on:

- Live Rinia network access
- Bridge support for major test networks
- Official explorer availability
- Wallet onboarding
- DonaSwap deployment and testing
- Contract verification and address transparency

---

## Near-Term Goals

Planned improvements include:

- Expanded documentation
- More developer tooling
- Improved onboarding flow
- Additional contract integrations
- Better ecosystem discoverability
- Broader application support

---

## Long-Term Direction

The long-term direction for FireChain includes:

- Growth of the on-chain ecosystem
- Increased liquidity and utility
- Expanded cross-chain interoperability
- Stronger infrastructure and tooling
- Broader user adoption
